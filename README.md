## 🛠️ Prerequisites & Setup

To run the notebooks and scripts in this repository locally, you will need to set up your environment.

### 1. Local Environment
Ensure you have the following installed:
* **Python 3.x**
* **PyTorch** (Installation depends on your OS and whether you have a CUDA-enabled GPU. See the [official guide](https://pytorch.org/get-started/locally/))
* Additional libraries: `torchvision`, `numpy`, `pandas`, `matplotlib`

You can install the core dependencies using pip:
```bash
pip install torch torchvision numpy pandas matplotlib
